# Panda's Louja

Projeto de desenvolvimento de uma loja virtual para vendas de produtos com e sem estoque. 


## Funcionalidades

- Temas dark e light
- Adição e remoção de produtos
- Controle de Estoque
- Review de vendas por mês
- Etc.
